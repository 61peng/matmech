# Material Mechanism Dataset

本项目是论文 _A Multimodal Dataset of Causal Mechanisms in Materials Science Literature_ 的完整仓库。

## 目录结构

按需填写各目录/文件的具体作用。

- instruction/ : 用于大模型生成的指令数据
- pre-test/: 正式实验前的测试脚本以及需要预先处理的数据
- models/ : 使用的模型权重
- pipeline/ : 用于可信生成材料四面体要素、微观图像功能以及机制知识的代码框架
- output_file/ :模型生成的数据，按照期刊名称划分，里面包含 raw/和 processed/
  - raw_data/ : 模型直接生成的数据，以 jsonl 文件存储，每一行代表一篇论文的数据
  - processed_data/ : 将大模型生成的内容解析为 json 格式并存储的数据
- final_output/ :使用 pipeline 生成的最终数据，创建这个文件的本意是把三阶段生成的内容放到一个文件里，更好的应用于标注系统
- matmech/ : 用于开源的数据文件夹
- matmech4figshare/ :将 matmech/压缩为 zip 文件并补充对数据说明、数据样例、数据元数据等信息

## 后日谈

在项目结束后的检查中，发现还是残留了一些不属于研究型论文的 pdf 输入进来，并最终由大模型生成了相应的四面体机制数据，这些数据是不准确的。

此外，还有还有撤稿论文，以及一些数据属于预印版本，里面的论文图片没有排版，都放到了论文的最后，从而干扰了论文数据的获取和解析，所以最好也进行清理。

我们从 pdf 层面对这些数据进行了清理，即这些文件被存放到 solved_pdf 的{journal}\_filter 路径下，在这些路径下的数据是 invalid 的。
